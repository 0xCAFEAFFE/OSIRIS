### MODULES
+ allways uppercase, e.g.: RTC_, ADC_

### VARIABLES
+ all variables start with a lowercase letter

SCOPE			FORMAT										EXAMPLE
stack/local		lower snakecase	                            temp_rx_str, counts_old
static/file		camelcase			                        txBuffer, secTickFlag
extern/global	module prefix, underscore, camelcase		RTC_upTime, RAD_totalDose

### FUNCTIONS
+ all functions start with an uppercase letter

SCOPE			FORMAT										EXAMPLE
private			pascalcase              					TxChar(), CalcSysTime()
public			module prefix, underscore, camelcase		UART_RxString(), RAD_GetDoseRate()

### DEFINES/ENUMS
+ all upper snakecase, e.g: UART_BAUD_RATE, RAD_CONV_FACTOR

### TYPEDEFS
+ modul prefix, underscore, pascalcase, '_t' appended, e.g: RTC_Time_t, RAD_AlarmRate_t;

### IF
+ the following is only allowed if BOTH controlled statements are single lines:

    if (condition) { singleLineOfCode(); }
    else { anotherSingleLine(); }

+ if one controlled statement does not fit in a single line, this style shall be used:

    if (condition)
    {
        firstLine();
        secondeLineRequired();
    }
    else
    {
        singleLine();
    }

### FILE STRUCTURE ###
+ header
+ includes
+ defines
+ macros
+ private variables
+ public variables
+ private function declarations
+ public function definitions
+ private function definitions
+ ISRs
